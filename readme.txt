Plugin created shortcode to insert YouTube videos with microdate on Schema.org for rich snippet

== Description ==

The plugin allows to speed up and simplify the process of partitioning a video from YouTube about writing articles.
Click on the button and fill in all fields. Click on the "Ok" button and you're done.
For the correct operation of the microdata, all the fields in the Add microdata section are mandatory for filling

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `insert-video-with-schemaorg-ivws` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
